首先，讲下基本的使用方法,大致包括以下几个内容

mkdir test

cd test

git init   //初始化，本地建立空仓

git add aa.c

git add bb.c

....

git add *   //添加文件到本地仓

git commit -m "描述本次提交"

git push  //最后将本地仓推到server上即可
